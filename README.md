#Finance Dashboard

This project is a finance dashboard built using React and Tailwind CSS. It allows users to track income and expenses, view their balance, and understand spending patterns through charts.
The application includes functionality to add, edit, and delete transactions. Users can also search, filter, and sort transactions based on different criteria. A basic role-based system is implemented where an admin can modify data and a viewer can only view it.
All transaction data is stored in localStorage so that it persists even after refreshing the page.
The UI is designed to be clean and responsive, so it works well across different screen sizes. The dashboard also handles cases where there is no data and displays appropriate messages.
The charts are built using Recharts to show income versus expenses and the balance trend over time.
To run the project, install dependencies using npm install and start the development server using npm run dev.
This project focuses on component-based architecture, state management using React hooks, and creating a simple and user-friendly interface.
